Question-1: What are the different ways to select an element in the DOM?

There are many ways to select element in the DOM. There are getElementById, getElementsByClassName, getElementsByName, getElementsByTagName, querySelector, querySelectorAll etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

You can get the text inside an element with innerText and you can get the entire HTML code with innerHTML.The textContent property sets or returns the text content of the specified node, and all its descendants.

Question-3: What is event delegation in the DOM?

Event delegation means going up from the target element like a bubble, looking for the parent.Which is used add addEventListener.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process of moving from the target element to its parent until it reaches the last parent.

Question-5: How do you create, add, and remove elements using JavaScript?

To create element I have to use document.createElement. To add element use .appendChild and remove element to use .remove
